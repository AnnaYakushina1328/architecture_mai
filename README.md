# Система заказа такси https://www.uber.com/
## Приложение должно содержать следующие данные:
- Пользователь
- Водитель
- Поездка

## Реализовать API:
- Создание нового пользователя
- Поиск пользователя по логину
- Поиск пользователя по маске имя и фамилии
- Регистрация водителя
- Создание заказа поездки
- Получение активных заказов
- Принятие заказа водителем
- Получение истории поездок пользователя
- Завершение поездки

## 1. РОЛИ ПОЛЬЗОВАТЕЛЕЙ
1. Пассажир (Passenger/User)
- Создает аккаунт в системе
- Создает заказы на поездки
- Просматривает историю своих поездок
- Получает информацию об активных заказах
2. Водитель (Driver)
- Регистрируется в системе как водитель
- Получает список доступных заказов
- Принимает заказы на выполнение
- Завершает поездки
3. Администратор системы (Administrator)
- Поиск пользователя по логину
- Поиск пользователя по маске имя и фамилии
- Управление пользователями и водителями
- Мониторинг системы
